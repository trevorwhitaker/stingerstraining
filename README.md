# stingerstraining

## To run locally:

Open a terminal on the root folder:

``` bash
npm run start:dev
```

In a new terminal:
``` bash
cd client-src
npm run start
```

In a browser, go to `http://localhost:3000` to start developing, all files are auto rebuilt on change.

## For production builds:

``` bash
cd client-src
npm run build
```
